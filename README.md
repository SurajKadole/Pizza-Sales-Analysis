🍕 Pizza Sales Analysis using Power BI
📌 Project Overview

This project focuses on analyzing pizza sales data using Power BI to uncover meaningful insights and support data-driven decision-making. The dashboard provides a clear view of sales performance, customer behavior, and revenue trends.

🎯 Objectives
Analyze total revenue, total orders, and average order value
Identify top-selling and least-selling pizzas
Understand customer ordering patterns
Discover peak sales days and times
Evaluate performance by pizza category and size
📊 Dashboard Features
Interactive and dynamic visualizations
KPI cards for key metrics
Time-based analysis (daily, weekly, monthly trends)
Filters and slicers for better data exploration
Category-wise and size-wise performance insights
🛠️ Tools & Technologies
Power BI
DAX (Data Analysis Expressions)
Microsoft Excel / CSV
📈 Key Insights
Peak sales occur during weekends and evening hours
Certain pizza categories consistently outperform others
A small number of products contribute to a large portion of revenue
Seasonal and time-based trends impact sales performance
📂 Dataset

The dataset includes:

Order details
Pizza types and categories
Quantity sold
Pricing and revenue data
